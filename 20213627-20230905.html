Linux基础命令和Git环境配置

1. Linux基础命令

    
ls命令
语法 ：ls[选项][目录或者文件]
功能：对于目录，该命令列出该目录的子目录和文件。对于文件列出文件名及其它信息
常用选项：-a 列出目录显得所有文件，包括以"."开头的隐藏文件	-l：列出文件的详细信息
pwd命令
语法：pwd
功能：显示用户当前所在目录
cd指令
语法：cd 目录名
功能：改变工作目录。将当前工作目录改变到只能指定工作目录下
cd ..  返回上一目录
cd -   进入最近访问的目录
cd ~   进入主工作目录
stat指令 
语法：stat 目录或者文件
功能：查看文件或者目录属性
touch [选项] 文件
功能：touch命令可以更改文档或者目录的日期文件，包括存取和更改时间，或者是创建一个文件。
mkdir指令
语法：mkdir[选项]目录名
功能：在当前目录下创建一个目录
 rm指令
语法：rm[选项]文件名或者目录
功能：删除文件或者目录
cp指令
语法：cp[选项]源文件或者目录 目标文件或者目录
功能：后面可以加路径，如果最后是目录，并且在该目录下不存在同名文件，将该文件拷贝到该目录下。同一目录中不能存在同名文件。
类似的指令还有很多比如mv指令 echo指令等因篇幅太长不多提及。都有了一些初步的认识以及应用。并且熟练运用文件的基础操作

2. 软链接和硬链接

    
2.对软链接，硬链接初步认识
软连接：它的功能是为某一个文件在另一个位置建立一个同步的链接。简单来说，就是window里面的快捷方式。
语法：ln -s target source
ln -s:表示创建一个软链接；
target：表示目标文件（夹)，即源文件
source:表示当前目录的软链接名，即被创建出来的软链接名称以及放置在何处。
硬链接：硬链接通过指向索引节点来进行连接（inode index）。在Linux中，多个文件名指向同一个索引节点是存在的。硬链接的作用是允许一个文件拥有多个有效路径名
这样用户就可以建立硬链接到重要文件，以防止“误删”的功能。
语法：ln  target source
